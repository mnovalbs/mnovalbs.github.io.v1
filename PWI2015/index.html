th,thead{
  text-transform:uppercase;
  font-weight:bold;
  background:red;
}
tr:nth-child(even){
  background:green;
}
  1. th,thead

    Selector untuk bagian header (thead) dan sel pada header (th).

  2. text-transform: uppercase

    Untuk membuat text menjadi uppercase (kapital)

  3. font-weight: bold

    Mengubah weight font, bisa bold, normal, dll atau bisa dalam bentuk angka. (200/500/700) dll

  4. background: red

    Mengganti warna background, bisa juga menggunakan background-color.

  5. tr:nth-child(even)

    Selector tiap tr(baris) pada tabel dengan urutan genap/even. Untuk ganjil menggunakan odd.

  //Inisiasi karakter captcha [a-zA-Z0-9]
  var karakter = "01234567890abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ";

  //Inisiasi captcha kosong dengan tipe String
  var captcha = "";

  //Mengisi captcha yang tadinya kosong ditambah dengan string dari karakter captcha paling atas (dengan random)
  //charAt fungsi untuk mengambil karakter dari string
  //Math.random() menghasilkan bilangan real random 0.xxxxx
  for(i=1;i<=4;i++){
    captcha += karakter.charAt((Math.random()*100)%karakter.length);
  }

  //Mengambil element dengan id "captcha" kemudian diisi dengan captcha
  document.getElementById("captcha").innerHTML = captcha;

  //Cek apakah captcha yang diinput sama dengan captcha yang ditulis
  function register(){
    var isiCaptcha = document.getElementById("isiCaptcha").value;
    if(isiCaptcha!=captcha){
      peringatanSalah("Captcha Salah");
    }
  }

  //Menampilkan peringatan
  function peringatanSalah(a){
    var tempatPeringatan = document.getElementById("peringatan");
    alert(a); //Menggunakan Alert Pop Up
    //tempatPeringatan.innerHTML += "<p>"+a+"</p>"; //Menggunakan text peringatan di bawah
  }

  1. oninvalid

    Atribut ini digunakan apabila input tersebut tidak diisi dengan sesuai / valid, dan akan melakukan fungsi yang diinginkan.

  2. required

    Atribut ini digunakan untuk input yang wajib disii.

  3. type

    Atribut ini digunakan untuk menentukan tipe yang dapat dimasukkan pada input. Pada jawaban di atas menggunakan url, email, password dan text.

  4. pattern

    Atribut ini digunakan untuk menentukan karakter apa saja yang dapat dimasukkan pada input.

    • [a-zA-Z ]+ : Karakter a-z, A-Z, dan karakter spasi.
    • https?://[a-zA-Z0-9.-]+\.[a-z]{2,5} : String berawal "http://" atau "https://"; kemudian karakter a-z,A-Z,0-9,strip, dan titik; kemudian karakter titik ditambah dengan karakter [a-z] minimal 2 sampai 5 karakter.
    • [a-zA-Z0-9]{6,} : Karakter a-z,A-Z,0-9 minimal 6 karakter
  5. onclick

    Berisi fungsi yang akan dieksekusi apabila element tersebut diklik.

var Circle = Object.inherit({
  initialize: function(radius){
    this.radius = radius;
  },
  area: function(){
    return Math.PI * this.radius * this.radius;
  },
  circumference: function(){
    return 2 * Math.PI * this.radius;
  }
});

var c = Circle.create(10);
alert(c.area());
alert(c.circumference());
  1. var c = Circle.create(10);

    Membuat objek Circle dengan mengisi radius 10

  2. alert(c.area());

    Menampilkan peringatan yang berisi kembalian nilai dari fungsi area. Dengan fungsi area mengembalikan nilai Math.PI * this.radius * this.radius. Dengan radius yang sudah diinisiasi di awal (10). Silahkan dihitung sendiri hehe

  3. alert(c.circumference());

    Sama seperti nomer 2
